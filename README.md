# Code-Compiler
## Features ##
* Support Multiple Languages(C,C++,Java,Python )

* Results
* Execution Speed/Time
* Error Message
* Support Multiple Questions
## client ##
cd client

npm install

npm start
## Server ##
python manage.py migrate

python manage.py makemigrations

python manage.py runserver
